Case Study: Data Center & Network Automation

Challenge: Automate provisioning for large-scale data center infrastructure across multi-subnet and multi-VLAN environments.

Our Solution: Leveraged Tinkerbell, PXE boot, and custom BYO DHCP to automate infrastructure bring-up. Designed scalable Kubernetes clusters (EKS & bare-metal) with advanced Cilium BGP and high-availability using Keepalived VIPs and both Active-Active and Active-Standby network models. Developed Day 0/1 Arista EOS templates (VRF, static routing, IP planning). Integrated DHCP relay, netplan, and FRR for robust underlay BGP peering. Automated all network and platform workflows via GitLab CI/CD pipelines, including deployment and IPAM documentation with Git-driven diagrams.

Result: Reduced manual deployment time by 70%, enabled rapid scale-up, and ensured compliance with security and documentation standards.
